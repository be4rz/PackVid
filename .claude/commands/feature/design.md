Invoke the `feature-design` skill to design the following feature:

<prompt>$ARGUMENTS</prompt>

Follow all 7 phases defined in the skill:
1. Discovery - Understand the feature idea
2. Market Research - Analyze market landscape
3. User Research - Understand target users
4. Product Definition - Define problem and value proposition
5. Business Flows - Visualize key processes
6. User Stories - Break into implementable stories
7. Go-to-Market - Plan launch strategy

Store all outputs in `agent-workspace/features/<feature-name>/design/`.
